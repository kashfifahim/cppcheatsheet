Notes on C++

Introductory Notes

  • C++ is a platform-dependent langauge.
  • A program written and compiled on a specific OS won't run on any other OS.
  • Structured programming language means that we can divide a program into different parts.
  • Statically-typed, variables are explicityly declared and determined at compile time.
  • Compiled language, programs cannot be executed without compilation.
  • STL, makes development faster and easier.
  • Pointers are used to interact with memory.
  • OOP, lowers the development cost by giving a clear structure to the program and allowing the reusability of code.
